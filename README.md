# Viya Experience : Forecasting
Vous êtes un "Demand Planner" et un "Data Engineer" dans une entreprise de grande distribution aux US, en charge de la prévision des ventes. Ces prévisions servent aux département Achat, Finance et surtout Supply Chain (gestion des flux et des stocks entre les entrepôts et les magasins).  ​

Vous avez à votre disposition les historiques de Ventes, de Prix ainsi qu’un calendrier d’événements. Les tables corresspondantes sont : 
- VXPF_VENTES_HEBDO
- VXPF_PRIX_HEBDO
- CALENDAR​

Vous allez :​
- Explorer les données​
- Préparer une table adaptée pour le moteur de prévision​
- Lancer plusieurs modélisations​
- Exporter la prévision finale​
